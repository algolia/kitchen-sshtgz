kitchen-sshtgz
==============

Improved file transfers for test-kitchen (especially EC2) using tgz archives and SSH.

Quick Start
-----------

Run `chef gem install kitchen-sshtgz` and then set your transport to `ssh_tgz`:

```
transport:
  name: ssh_tgz
```
